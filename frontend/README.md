# HyroX Analytics Frontend

Next.js frontend application for HyroX Analytics.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
cp .env.example .env.local
# Edit .env.local with your configuration
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Dashboard**: Overview of trading performance
- **Trading Journal**: Track and manage trades
- **Profit Calculator**: Calculate payouts and scaling eligibility
- **Wallet Integration**: Connect with Phantom or Solflare wallets

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Solana Web3.js
- Recharts
- React Query
